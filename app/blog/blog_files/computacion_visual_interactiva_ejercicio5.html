Utilizando el ejemplo del cubo, se implementó una interfaz grafica simple que permite al usuario cambiar la perspectiva de la camara.

Para cambiar la perspectiva se utiliza mat4 para manipular la matriz de la camará con una translación.

Se definieron 3 variables para definir la translación en "x", "y" y "z".

Dado que la cámara se actualiza cada vez que se dibuja un fotograma, al modificar las variables se puede observar el cambio inmediatamente.

Al presionar cada botón se modifican las variables a un valor determinado. Para este caso, se modifica la variable z para modificar la distancia entre la camara y el objeto.

Por ultimo, el usuario puede modificar manualmente los valores de las variables. Para esto debe indicar los valores [x,y,z] utilizando los input y presionar el botón "Set"
